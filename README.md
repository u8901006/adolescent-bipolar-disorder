# Adolescent Bipolar Disorder Research Daily Report

AI-curated daily literature monitoring for adolescent bipolar disorder research.

## Features

- Automatically fetches latest papers from PubMed using comprehensive search strategies
- Uses Zhipu AI (GLM-5-Turbo) to analyze, summarize, and categorize research
- Generates beautiful HTML reports deployed to GitHub Pages
- Tracks processed papers to avoid duplicate summaries
- Runs daily at 22:05 GMT+8

## Search Scope

Based on the Adolescent Bipolar Disorder Research Toolkit, covering:
- Diagnosis, nosology, and differential diagnosis
- Pharmacotherapy and medication safety
- Psychotherapy and family interventions
- Neuroscience, neuroimaging, and biomarkers
- Genetics and family risk
- Epidemiology and longitudinal course
- Social determinants and school functioning
- Suicide and self-harm
- Sleep and circadian rhythm

## Tech Stack

- Node.js 24
- PubMed E-utilities API
- Zhipu AI API (GLM-5-Turbo → GLM-4.7 → GLM-4.7-Flash fallback chain)
- GitHub Actions with SHA-pinned actions
